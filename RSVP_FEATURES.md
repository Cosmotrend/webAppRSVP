# ✨ Semilac Days RSVP App - Premium Features

## 🎨 Design Philosophy
Glamorous & luxurious with dynamic & energetic animations - created for an unforgettable experience.

## 🚀 Premium Features Implemented

### 1. **Morphing Aurora Backgrounds**
- 3 animated gradient blobs that move independently
- Smooth morphing animations with different speeds
- Radial gradients creating ethereal glow effects
- Pink, rose, and gold color palette

### 2. **Magnetic Button Hover Effects**
- Buttons follow your cursor within proximity
- Spring physics for natural movement
- Sound feedback on hover
- Scale animations on press

### 3. **Sound Feedback System**
- Click sounds for interactions
- Success chimes for confirmations
- Spinning sound for wheel
- Victory fanfare for winning
- Hover sounds for premium feel

### 4. **Animated Confetti Celebrations**
- Multi-angle confetti burst on confirmation
- Epic 5-second celebration on prize win
- Custom pink/gold color scheme
- Physics-based particle system

### 5. **Physics-Based Wheel Spinner**
- Realistic spin with easing
- 5+ full rotations before landing
- Animated center button
- Visual feedback with pointer
- Click anywhere on wheel to spin

### 6. **Parallax Scrolling Effects**
- Floating particles on greeting screen
- Layered aurora backgrounds
- Depth through motion

### 7. **Micro-Interactions**
- Input focus animations
- Button scale on tap
- Rotating logo on hover
- Pulsing badges
- Smooth page transitions

### 8. **Premium Typography**
- Cormorant Garamond for elegant titles
- Montserrat for UI elements
- Gradient text effects
- Custom letter spacing

### 9. **Glassmorphism UI**
- Backdrop blur effects
- Translucent cards
- Layered depth
- Border glow effects

### 10. **Mobile-First Design**
- Optimized for 375×812 viewport
- Touch-friendly interactions
- Smooth scrolling
- Responsive layouts

## 📱 User Flow

1. **RSVP Form** → Fill details with animated inputs
2. **Confirmation** → Confetti celebration + ticket number
3. **Wheel Code Entry** → Enter ticket code with validation
4. **Personal Greeting** → Animated welcome with user's name
5. **Wheel Game** → Interactive spinning wheel
6. **Prize Result** → Epic confetti + coupon code

## 🎭 Animation Highlights

- **Entry animations**: Staggered reveals with spring physics
- **Hover states**: Magnetic attraction + sound feedback
- **Transitions**: Smooth page-to-page navigation
- **Celebrations**: Multi-layered confetti explosions
- **Ambient motion**: Continuous background animations

## 💾 Data Management

- LocalStorage for ticket data
- No backend required
- Session persistence
- Prize tracking

## 🎵 Sound Design

All sounds generated using Web Audio API:
- Click: 800Hz sine wave
- Success: Dual-tone harmony
- Spin: Frequency sweep
- Win: 4-note ascending scale
- Hover: Quick ping

## 🌈 Color Palette

- Primary Pink: `#F8A4C8`
- Rose: `#c47090`
- Light Pink: `#ffdaec`
- Gold: `#D4A574`
- Dark Background: `#0D0008`

- React + TypeScript

- Canvas Confetti for celebrations
- Web Audio API for sounds

- Lucide React for icons
